## BTC -> DAI -> USD

### This is a program to check the loss / gain of transfering different ammounts of BTC to DAI through hitbtc

### installation

    - clone repo locally

    npm install

### To run

    cd src

    node hitbtc_orders.js

    node btc_price.js
